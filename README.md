# Skeleton
Start without delay coding your app.

You may need to install redis.

1. Install dep, a [basic dependency tracking](https://github.com/cyx/dep) gem.
2. Run `dep install` to install the necessary gems.
3. `compass watch ./`
4. Fire up `shotgun`

Includes:
* Shotgun
* Cuba
* OHM
* normalize.css
* Compass
